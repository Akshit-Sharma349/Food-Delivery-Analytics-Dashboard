# Food Delivery Analytics Dashboard | Power BI

## Project Overview

This project is an end-to-end business analytics dashboard built using Power BI to analyze the performance of a food delivery business.

The project focuses on three major business areas:

* Revenue Analytics
* Customer Analytics
* Delivery Operations

The dashboards were designed to simulate real-world business reporting used by food delivery and marketplace companies to monitor KPIs, customer behavior, operational efficiency, and revenue performance.

---

# Objectives

The main objectives of this project were:

* Analyze revenue and order trends
* Identify high-performing cities and cuisines
* Understand customer behavior and segmentation
* Monitor delivery performance and customer ratings
* Build executive-level business dashboards using Power BI

---

# Tools & Technologies Used

* Power BI
* DAX (Data Analysis Expressions)
* SQL
* Data Modeling
* Business Analytics
* Data Visualization

The analytical backend was built using SQL for KPI generation, customer analytics, cohort analysis, operational reporting, and Power BI dataset preparation.
